# Simple Chat Messaging Simulator

## How to Run
1. Install Python 3
2. Run: python main.py

## Features
- Chat GUI
- Auto incoming messages
- Styled messages (Decorator pattern)
- Notifications (Observer)
- ChatEngine Singleton
- MessageFactory + ChatSessionBuilder
